## 수강생용 슬라이드의 구성 원칙

각 회차는 실무자가 **AI 툴을 실제로 어떻게 다루고 어떤 방향으로 쓸지**에 초점을 맞췄습니다.

- **학습 목표 → 오늘의 흐름(120분) → 개념 → 「오늘의 툴 다루기」 → 실습 → 핵심 3줄 → 미션 안내** 순서로 흐릅니다.
- 「오늘의 툴 다루기」 카드는 도구별로 *이렇게 쓴다 / 무엇을 보나 / 주의점*을 명시해, 툴을 "답 얻는 기계"가 아니라 "브랜드를 측정·개선하는 계측기"로 다루도록 설계했습니다.
- 모든 실습·프롬프트는 그대로 복사해 쓸 수 있는 형태입니다.

## 강사용 가이드의 구성 (수강생 배포 금지)

강사가 수업을 그대로 운영할 수 있도록, 회차마다 다음을 담았습니다.

1. **운영 타임라인 × 시연 매핑** — 분 단위 진행표에 시연 번호(D1~D5) 연결
2. **시연 시나리오 D1~D5** — 강사가 보여줄 예시 화면을 유형별로 분류
   - `웹검색` · `사이트` · `유튜브` · `뉴스` · `라이브 시연` 태그로 구분
   - 각 시연마다 **링크/검색어**와 **"이 말을 하세요" 시연 포인트**를 명시
3. **핵심 시연 상세** — 하이라이트 시연의 진행 순서
4. **시연 백업 플랜** — 라이브 실패·결과 변동 시 대응 (실패를 교육 소재로 전환)
5. **미션 배포문** — 공유 시트에 그대로 게시할 수 있는 과제 안내문
6. **미션 평가 루브릭** — 항목·배점·우수/미흡 기준
7. **토론 유도 질문 & 예상 질문 대응** — 강사 답변 가이드 포함
8. **수업 전 준비 체크리스트**

### 시연 자료 유형 태그

| 태그 | 의미 | 예시 |
|---|---|---|
| `웹검색` | 검색엔진에서 라이브 검색 | 구글 AI Overviews, site: 검색, PAA |
| `사이트` | 특정 웹사이트/도구 화면 | Perplexity 출처, 공식 문서, 서치어드바이저 |
| `유튜브` | 유튜브 영상/채널 | Google Search Central 채널 |
| `뉴스` | 뉴스 검색·기사 | 제로클릭 검색, AI 검색 트래픽 기사 |
| `라이브 시연` | AI 툴 실시간 실행 | 프롬프트 실행, 라이브 리라이트 |

> 강사용 슬라이드의 모든 외부 링크·검색어는 **수업 전날 접속·노출 여부를 반드시 확인**하세요. AI 검색 결과는 날짜·계정·지역·도구에 따라 달라집니다. 백업 스크린샷은 필수입니다.

## 편집 방법

- 파일은 표준 `.pptx`로, PowerPoint · Keynote · Google Slides · LibreOffice Impress에서 모두 열립니다.
- 슬라이드 크기: 16:9 (13.333 × 7.5 inch)
- 기본 한글 폰트: 맑은 고딕 (없는 환경에서는 시스템 대체 폰트로 렌더링)
- 브랜드에 맞게 색/로고를 바꾸려면 슬라이드 마스터가 아니라 각 도형 색을 직접 조정하세요.

## 재생성 (선택)

슬라이드는 `python-pptx`로 코드 생성했습니다. 내용 수정 후 재빌드하려면:

```bash
pip install python-pptx
# 수강생용
OUT_DIR="./수강생용" python3 build_student.py
# 강사용
OUT_DIR="./강사용" python3 build_instructor.py
```

*(빌드 스크립트 `helpers.py` · `build_student.py` · `build_instructor.py`는 별도 보관)*
