# 전수검사 (베놈 검색정보 진단 프로젝트)

`hospital-marketing/` 전체를 아래 10개 영역으로 점검한다. **모든 사항 사전 승인** 상태로 실행되므로
확인 질문 없이 진행하되, 자잘한 문제(오탈자·깨진 링크·명백한 코드 정리)는 즉시 고치고,
판단이 필요한 문제(법무·구조 변경)는 보고서에 "결정 필요"로 분류한다.
마지막에 반드시 `hospital-marketing/docs/전수검사-보고서-YYYY-MM-DD.md`로 결과를 저장하고 커밋한다.

## 점검 영역

### 1. SEO
- 공개 페이지(landing) 기준: title/description 길이, canonical, Open Graph, JSON-LD 유효성(파싱 테스트),
  단일 h1, 시맨틱 헤딩 계층, sitemap.xml·robots.txt 일치 여부
- 내부 페이지(대시보드·관리자단·입지·처리방침)는 noindex 유지 확인
- Playwright로 렌더 후 `document.title`, 메타, JSON-LD를 실측 검증 (파일 grep만으로 끝내지 않기)

### 2. 의료법·법무 문구
- 금지 표현 grep: "순위 추적", "상위노출", "보장", "실시간 순위", "OO보다", "최고", "1등",
  "효과", "치료경험담", "전후사진" — 매칭 시 문맥 확인 후 위반 여부 판단
- 필수 고지 존재 확인: 각 화면·리포트에 "내부 참고용", "전재 금지", "당사 산식", "보장하지 않습니다",
  출처·기준일·측정 조건
- "미노출" 정의("API 응답 상위 5건 밖") 표기 여부
- 기준 문서: docs/법무-검토-메모-v2-결과.md 8장(출시 전 필수 조치 10건)과 대조

### 3. 콘텐츠 조건 충족
- 신청폼: 필수 2 + 선택 4 동의 분리, 선택 거부 시 제출 가능, 상세 내용 접이식 표시
- 리포트: 점수 명칭 "검색정보 운영 진단 점수(당사 산식)", 스냅샷 열 없음, 리뷰/사진은 병원 직접 입력 표기
- 처리방침: 10개 조항 + 쿠키(GA) 항목, placeholder 목록 정리

### 4. 오탈자·문구 일관성
- 한국어 오탈자, 띄어쓰기, 용어 일관성(예: "진단 점수" vs "경쟁력 점수" 혼용 — 후자는 금지),
  브랜드 표기 통일("(주)베놈", "venomad")
- 남아있는 구 브랜드("메디랭크") 흔적 검색

### 5. 링크 매칭
- 모든 상대 링크(a href, fetch 경로)가 실제 파일로 해석되는지 스크립트로 검증
- 앵커(#) 대상 존재 여부, 외부 링크 프로토콜(https)

### 6. 이미지·사진 정리 (webP 포함)
- `<img>`, background-image, 첨부 파일 전수 수집 → 사용처·개수·형식(webP 여부)·용량 표로 정리
- 래스터 이미지(jpg/png)가 있으면 webP 변환 권고 + `pipeline/convert_webp.py`로 변환
- 실사 사진이 없다면 "0장 — 전부 SVG/CSS 벡터"로 명시 보고

### 7. 코드 최적화 (간소화)
- 페이지 간 중복 CSS/JS(esc 함수, 게이지, 팔레트 변수) 식별 → 공용화 제안 또는 적용
- 죽은 코드, 미사용 CSS 클래스, 과도하게 복잡한 함수 단순화
- 파이프라인: 단위 테스트 통과 확인(python3 -m unittest discover -s tests) 후 리팩터링

### 8. UI/UX
- Playwright로 전 페이지를 데스크톱(1200px)·모바일(390px) 렌더 → 스크린샷 검토:
  레이아웃 깨짐, 텍스트 겹침, 표 가로 스크롤, 터치 타깃 크기, 다크모드
- 콘솔 JS 에러 0건 확인

### 9. 보안
- docs/보안-점검-체크리스트.md 항목 재검증: CSP 존재, 이스케이프 적용 지점, .env 미커밋,
  robots/noindex, 관리자단 배포 경고
- 새로 추가된 코드에 innerHTML 비이스케이프 삽입이 없는지 grep

### 10. 개인정보처리방침·이용약관(제공조건) 미비점
- 처리방침 placeholder 잔여 목록, 시행일, 수탁자
- 무료 진단자료 제공조건 7개 항목 존재 확인
- 법무 문서(docs/) 간 상호 모순 여부
