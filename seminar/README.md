# AI 한의원 검색점유 패키지 — 세미나 자료

한의원 원장 대상 세미나에서 "AI 한의원 검색점유 패키지"를 판매하기 위한 자료 모음입니다.
업로드된 상품 기획서, 판매 랜딩 원고, 가격/상담 스크립트, 월간 리포트 템플릿, 홈페이지 템플릿 문서를 토대로 제작했습니다.

## 구성

| 파일 | 용도 |
| --- | --- |
| `01_상품판매기획서.md` | 세미나 판매 기획서 — 아젠다, 핵심 메시지, 전환 전략, 예상 질문 대응, 성공 지표, 체크리스트 |
| `02_상품안내서.md` | 원장님 배포용 상품 안내서 (원문) |
| `상품안내서_인쇄용.html` | 상품 안내서 A4 인쇄용 3페이지 브로슈어 — 브라우저에서 열고 인쇄(PDF 저장) |
| `seminar_deck.html` | 세미나 슬라이드 덱 12장 — 브라우저에서 열고 방향키/클릭으로 진행, `#슬라이드번호`로 바로 이동 |

## 수정 방법

- 이미지 문구 수정: `images/*.svg`를 텍스트 에디터로 수정 후 브라우저로 열어 확인 (PNG는 SVG를 1600×900으로 캡처한 것)
- 슬라이드 텍스트: `seminar_deck.html` 안의 각 `<section class="slide">` 수정
- 모든 예시 수치(언급률 등)는 이해를 돕기 위한 예시이며, 노출 보장 표현은 사용하지 않습니다 (의료법 제56조 의료광고 기준 준수)
