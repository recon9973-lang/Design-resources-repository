# (주)베놈 병원 검색정보 운영 진단 프로그램

병원 전문 광고대행사 **(주)베놈(venomad)** 이 병원에 **무료로 배포**하는 검색정보 운영 진단 리포트/대시보드입니다.
반경 500m~2km 안의 경쟁 병원, 인구/상권 참고지표, 네이버 공식 API 기준 키워드 노출 등급을 종합해
**검색정보 운영 진단 점수(0~100, 당사 산식)** 와 정보 정비 제안을 제공합니다.
베놈은 이 자료를 광고 컨설팅의 근거 자료로 활용합니다 (리포트 자체는 병원 내부 참고용 · 광고물 전재 금지).

> 사업 모델·문구는 법무 검토(v1/v2, `docs/`)를 반영했습니다: 판매 없음 → 무료 배포 + 신청 기반 리드젠,
> "순위" 표현 금지, 스냅샷·플레이스 자동 수집 보류, 필수/선택 동의 분리.

## 프로토타입 미리보기

GitHub Pages 배포 시: `https://recon9973-lang.github.io/Design-resources-repository/hospital-marketing/`

화면 5개 (기획서 5장):

1. **대시보드 홈** — 경쟁력 점수(노출 40% · 경쟁밀도 25% · 수요입지 20% · 플레이스품질 15%), 반경 선택, 6개월 추이, 이번 달 액션 3가지
2. **지도 분석** — 반경 링 + 경쟁 병원 분포 (같은 진료과/유사 진료과 구분)
3. **키워드 노출** — 공식 API 지표와 월간 참고 스냅샷 분리 표기, 상위권/중위권/미노출 등급
4. **경쟁 벤치마크** — 리뷰 수·평점·사진 수 등 공개 집계값 비교
5. **액션 리포트** — 의료광고 리스크를 제거한 운영·정보 정비 중심 제안 + PDF 저장(인쇄)

## 설계 원칙 (B-safe)

- 공식 API·공공데이터 중심. 실시간 순위 추적, 대량 크롤링, 캡차/차단 우회 없음
- 실제 검색화면 스냅샷은 법무 검토 후 월 1회 참고용으로만, 원본 증빙은 관리자단 전용
- 고객 화면에는 등급화된 안전 요약값만 노출 (`internal_evidence_items.visibility_scope`, DB 트리거로 강제)
- 모든 화면·PDF에 데이터 출처, 기준일, 측정 조건, 디스클레이머 표기
- 추천 액션은 의료법상 안전한 운영 개선 중심 (효과 보장·후기 활용·비방 비교 제외)

## ⚠️ 대표님(베놈)이 직접 해야 할 일 — 최신 체크리스트

코드로 대신할 수 없는 항목입니다. 위에서부터 순서대로 하시면 됩니다.

### 배포 전 (필수)

1. **변호사 최종 컨펌** — 검토 메모 2건은 "초안"입니다. 완성된 실물(신청 페이지, 리포트 PDF,
   처리방침)을 첨부해 최종 확인을 받으세요. 질의 목록은 `docs/법무-검토-설명자료.html`(v2) 6장에 정리되어 있습니다.
2. **개인정보처리방침 빈칸 채우기** — `landing/privacy.html`의 {{수탁자}}(문자·메일 발송 업체),
   {{개인정보 보호책임자}} 이름·연락처, 시행일. 업체 계약 시 표준 약관(DPA) 동의 문서 보관.
3. **신청폼 접수처 결정** — 구글폼/Airtable/자체 서버 중 선택해 알려주시면 연동해 드립니다.
   법적 요건: 동의 일시·IP·동의 문구 버전 저장, 수신거부(suppression) 목록 관리.
4. **API 키 개인 보관** — 발급받은 3종 키(공공데이터·네이버·SGIS)는 세션 종료 시 서버에서
   사라집니다. 각 포털 마이페이지에서 재확인 가능하지만 메모해 두세요.

### 운영 정착 (배포 후 1개월 내)

5. **내부 프로세스 2건 문서화** — ① 광고물 제작 체크리스트(의료법 56조 금지표현·심의 대상
   여부·병원 승인자·리포트 수치 전재 금지) ② 재연락 금지 목록 운영 규칙.
   콜드콜 스크립트·문자/메일 템플릿은 `docs/무료진단-신청폼-동의서-초안.md` 7~8장 참고.
6. **첫 배포 대상 선정** — 콜드 문자·메일 금지(법무 의견). 기존 거래처와 신청 유입 위주로.
   지역·경쟁 밀도가 다른 병원을 섞으면 점수 변별력 검증도 겸할 수 있습니다.
7. **광고성 수신 동의 2년 주기 재확인** 일정 등록 (정보통신망법 시행령 제62조의3).
