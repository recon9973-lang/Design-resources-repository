병원 검색 노출 경쟁력 진단 SaaS (가칭 메디랭크 로컬)

작성일 2026-07-02 · 작성 [운영자 이름] · 첨부: 법무 검토 의뢰서, 기획서 전문, 샘플 리포트 PDF

1. 서비스가 하는 일 (한 문단)

운영 중인 병원의 원장이 병원 이름·주소·진료과를 등록하면, ①공공데이터로 반경 500m~2km 안의 경쟁 병원 수와 분포를 계산하고, ②네이버 공식 오픈 API로 핵심 검색 키워드에서 병원의 노출 여부를 확인하고, ③통계청 인구·상권 지표를 결합해 "마케팅 경쟁력 점수(0~100)"와 월간 개선 제안을 담은 대시보드·PDF 리포트를 제공하는 월 구독 서비스입니다. 고객은 병원(의료기관)이며, 환자를 상대하지 않습니다.

4. 현재 하고 있는 것 vs 계획 중인 것 (검토 대상)

지금 (MVP) — 이미 구현·운영 예정

  • 공식 API 3종 호출 (각 기관 발급 키, 한도 내 저빈도)
  • 월 1회 배치 수집 → 월간 리포트
  • 리뷰 수·평점 등 공개 집계값 표시 계획 (리뷰 본문 미수집)

계획 (법무 의견 필요) — 승인 전 미운영

  • 네이버 실제 검색화면의 월 1회 "참고 스냅샷" (5쪽 상세)
  • 네이버 플레이스 공개 집계값(리뷰 수·사진 수) 수집 방식
  • 약관·개인정보처리방침 제정

6. 고객에게 보여주는 것 / 절대 보여주지 않는 것

✓ 고객 화면·PDF 노출

  • 가공된 점수·등급·변화 추이
  • 반경 내 경쟁 병원 수, 병원명·거리(공공데이터 원문)
  • 키워드별 등급화 결과
  • 출처·기준일·측정 조건·면책 고지
  • 운영·정보 정비 중심 개선 제안

✕ 내부 전용 (관리자단·접근통제·감사로그)

  • API 원본 응답(payload)
  • (향후) 검색화면 스냅샷 원본 이미지·HTML
  • 수집 실패/차단 로그, 수집 조건 메타데이터
  • 경쟁 병원 광고 문구 원문, 리뷰 본문(수집 자체를 안 함)
  • 법무 검토 상태·내부 메모
이 분리는 데이터베이스 스키마 수준에서 강제됩니다: 내부 증빙 테이블(internal_evidence_items)은 법무 승인(approved) 전에는 고객 노출 등급으로 변경이 불가능하도록 DB 트리거로 차단하고, 원천자료 열람·다운로드·삭제는 감사 로그(audit_logs)에 기록됩니다. 권한은 고객/대행사/데이터운영/법무/관리자 5단계로 분리됩니다.

7. 하지 않는 것 (제품 정책으로 배제)

  • 실시간 순위 추적, 대량·매일 크롤링, 검색화면 자동 대량 수집
  • 캡차·차단 우회, 로그인 세션을 이용한 수집, 개인화된 검색 결과 이용
  • 리뷰 본문·환자 치료경험담 수집 및 분석
  • "상위노출 보장", "순위 상승 약속", 치료 효과 보장 표현
  • 특정 경쟁 병원 비방 또는 단정적 비교우위 표현
  • 환자 개인정보 수집 (고객사 담당자 계정 정보만 취급)

8. (검토 대상) 월간 참고 스냅샷 운영안

공식 API 결과와 실제 검색화면은 다를 수 있어, 리포트 신뢰도를 위해 월 1회에 한해 실제 검색화면을 확인·기록하는 방안을 검토 중입니다. 법무 의견 전에는 운영하지 않습니다.

항목운영안
빈도·규모병원당 핵심 키워드 10~30개, 월 1회 (사람이 직접 검색하는 수준의 접근량)
방식1안: 직원 수작업 검색·기록 / 2안: 저빈도 반자동(스크립트 보조) — 두 안의 법적 차이 질의
조건 기록기준 위치·시간·기기·비로그인 여부를 함께 저장 (재현 가능성 확보)
금지 사항캡차/차단 우회 없음 · 로그인/개인화 결과 사용 없음 · 부하 유발 없음
보관원본 화면은 내부 증빙 전용(6쪽 접근통제 체계) · 고객에게는 "참고 스냅샷" 표기의 등급 결과만
표시"실시간 순위" 아님을 명시, 공식 API 지표와 분리 표기

9. 의료광고 관련 문구 정책 (리포트 내 제안 문구)

허용하는 제안 예시

  • "플레이스 진료 카테고리를 실제 진료 항목과 일치시키세요"
  • "진료시간·주차·예약 링크를 최신화하세요"
  • "해당 진료를 실제 제공하는 경우에만 소개·FAQ에 반영하세요"
  • "합법적인 만족도 조사·재방문 안내 절차를 정비하세요"

배제하는 제안 예시

  • 치료 효과·기간 보장 표현
  • 환자 후기·전후사진을 광고 소재로 활용하는 제안
  • "OO병원보다 낫다" 류의 비교·비방
  • 대가성 리뷰 유도

모든 리포트에 고지: "본 리포트는 마케팅 운영 참고용이며 특정 검색 순위 달성이나 매출 증가를 보장하지 않습니다. 개선 제안은 의료법상 의료광고 규정을 준수하는 운영·정보 정비 활동으로 한정됩니다."
