# 컨설팅 소견 설계 — 원장 궁금증 → 환자 유입 해답 맵핑 (2026-07-04)

배경: 진단 리포트가 "숫자"에서 끝나면 원장은 움직이지 않는다. 원장이 실제로 밤에 하는
고민과 대행사에 물어보고 싶은 질문에 리포트가 직접 답해야 DB 전환(상담 신청)으로 이어진다.
이 문서는 리포트의 **컨설팅 소견 섹션** 설계 근거와, 진단 결과를 원장 언어로 옮긴 맵핑을 정리한다.

> 법무 정합(v2): 무상 배포 · 콜드콜 금지 · 리뷰 본문 미수집 · "보장/최고/완치" 등 과장 표현 금지.
> 아래 해답도 전부 이 원칙 안에서 작성. 가짜 리뷰·순위 보장 등은 오히려 리스크로 명시.

## 1. 원장 페르소나 (왜 다른가)

- 대행사 불신이 기본값("돈만 날렸다"). → 우리가 고충을 안다는 걸 진단으로 증명.
- 의료법 리스크에 극도로 민감(보건소·영업정지). → 안전을 먼저 말한다.
- 경영자라 숫자·ROI·리스크에 반응. → 트래픽이 아니라 전환·비용으로 말한다.
- 시간 극빈. → 3분 안에 "내 상태"와 "뭘 하면 되는지"가 보여야.

## 2. 리포트에 구현한 구조 (auto_diagnose 리포트)

1. **노출 한눈에 보기** — 종합/플레이스/콘텐츠 노출률(색 구분 링게이지), 영역·지역 단계 막대.
2. **키워드 매트릭스 + 영역별 상위 5** — 미노출이어도 "그 자리에 누가 떴는지" 노출.
3. **입지 참고 분석** — 반경 개략도 + 속도계 게이지(낮음/보통/우수) + 구간 막대.
4. **컨설팅 소견 — 지금 무엇을 해야 하나** (신설)
   - ① 이번 달 우선 개선 액션(번호 + 근거)
   - ② 원장님이 궁금해하는 것 → 해답(Q&A 6종)
   - ③ CTA: 리스크 진단·검색여정 점검·무료 정밀검진(모두 무료·콜드콜 없음)
5. **측정 기준** — 항목/기준 표 + 고지(내부 참고용·전재 금지·보장 없음).

## 3. 원장 궁금증 → 해답 맵핑 (리포트 Q&A의 근거)

| 태그 | 원장의 질문(Q) | 해답(A) — 환자를 찾아오게 하려면 |
|---|---|---|
| 유입 vs 전환 | 플레이스엔 뜨는데 왜 신환이 안 늘까? | 노출(트래픽)≠내원(전환). 유입 뒤 콘텐츠 신뢰도·리뷰·예약 동선에서 이탈. 노출 숫자보다 "예약까지의 길"을 점검 |
| 검색 여정 | 환자를 어디서 놓칠까? | 증상①→지역·치료 비교②→병원명 검증③. 리포트의 지역단계로 ②, 브랜드로 ③ 확인. 빈 단계를 채움(증상①은 콘텐츠 보강) |
| 콘텐츠 공백 | 블로그·카페가 왜 중요? | 비교·검증 자리. 상위5에서 누가 찼는지 보고, 광고성 글 대신 진료 사례 스토리텔링으로 채움 |
| 의료법 안전 | 공격적으로 하고 싶은데 보건소가 무섭다 | 문구를 리스크 진단에 넣어 금지표현 선제 제거. 안전 범위가 결국 오래감 |
| 리뷰 관리 | 가짜 영수증 리뷰 사도 되나? | 필터링 삭제·위반 대상. 자발 리뷰 동선 + 악성 리뷰 답글 운영이 더 효과적 |
| 차별화 USP | 우리만의 강점 표현법? | "친절·최신장비"는 다 씀. 원장 진료철학·특정 시술 경험을 콘텐츠 축으로 |

## 4. 채널별 원장 속마음 → 타격 전략 (콘텐츠 축·비측정 영역 포함)

측정 가능한 영역(플레이스·블로그·카페·웹문서·뉴스·이미지·지식iN)은 리포트가 데이터로 보여주고,
공식 API로 관찰 불가한 영역(파워링크·브랜드콘텐츠 등 광고)은 **전략 가이드**로만 다룬다(법무: 화면수집 보류).

- 플레이스: "순위 올려준다"=불법 어뷰징 위험. 정공법(체류·예약연동·톡톡·텍스트) 락인.
- 파워링크(광고): 대표키워드 집착 탈피, 세부질환+치료법 재배치·부정클릭 방어. (자사 성과는 검색광고 API로 합법 연동 가능)
- 홈페이지: 7초 이탈 = 진료철학·USP 부재. 단일 랜딩(증상공감→과정→결과→예약).
- 블로그: 2026 심의 강화 고지 + 심의 통과형 스토리텔링 대필.
- 유튜브: 진료실 Q&A 쇼츠, 원장 시간 월 1시간.
- 카페·지식인: 그레이 바이럴 금지, 실만족 고객 앰배서더.
- SNS: 과별 포지셔닝(비급여=비포애프터/이벤트, 급여=진료철학·건강꿀팁).
- 리뷰: 가짜 금지, 자발 동선 + 명품 답글 템플릿.

## 5. DB 전환 시나리오 (콜드콜 없이)

1. 무료 자가진단·리포트로 "위기감(의료법·비용)" + "해답" 인지.
2. 결과 하단 CTA: 리스크 진단·검색여정 점검·무료 정밀검진 신청(필수 동의만).
3. 신청 시 차트룸(관리자)에 점수·등급 실시간 인입.
4. 후속은 **동의 채널이 있을 때만** — "의료법 소지 높은 글 3개 무료 피드백" 식 가치 제안.

## 6. 다음 개선 여지

- 기준선 50점 → 업종·지역 실측 평균으로 교정("동종 대비 상위 %").
- 증상 키워드(①정보탐색) 측정 편입(현재 지역+진료과 중심).
- 자사 파워링크 성과 합법 연동(광고주 계정).
- 리뷰 건강도: 본문 미수집 원칙 유지 → 체크리스트·직접입력 우회.
