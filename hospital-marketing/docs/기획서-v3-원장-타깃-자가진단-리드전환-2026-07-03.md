# 기획서 v3 — 원장 타깃 자가진단 · 리드 전환 (SIP 전략)

정의: 이 프로그램은 단순 웹 기능이 아니라, 원장에게 **위기감(의료법·비용 낭비)**을 주고
**해결책(베놈의 전문성)**을 갈구하게 만드는 **온라인 영업 사원(SIP, Sales in Print)**이다.
v2(서비스 정의·페르소나)를 계승하고, 원장 페르소나 심화 전략을 실행 설계로 구체화한다.

> ⚠️ 톤 균형: 베놈은 무상 배포·이미지 제고가 목적(v2 정책)이다. "위기감" 프레임은
> **정확한 사실 고지**로 구현하며, 공포 마케팅·과장·"보장" 표현은 쓰지 않는다. 의료법·
> 개인정보 리스크를 지적하는 도구가 스스로 그 리스크를 범하면 신뢰가 붕괴된다.

## 1. 원장 페르소나 — 왜 다른가

개원의는 일반 기업 마케팅 담당자와 완전히 다르다.
- **극심한 대행사 불신**: "광고 대행사에 돈만 날렸다"가 기본 전제.
- **의료법 리스크 극도 민감**: 보건소 신고·영업정지 = 생계 위협. 본인이 카피 하나까지 검수.
- **경영자 마인드**: 숫자·ROI·논리적 리스크 분석에 강하게 반응. "우리가 최고"엔 냉담.
- **시간 극빈**: 진료로 하루가 꽉 참. 3~4클릭 안에 가치를 못 주면 이탈.

핵심 전략: "우리가 잘한다"가 아니라 **"우리는 원장님의 고충을 이만큼 깊이 안다"**를
진단형 콘텐츠로 증명한다.

## 2. 밤잠 설치는 진짜 고민 3가지 (Pain Points)

| # | 고민 | 원장의 시선 | 우리의 어필 포인트 |
|---|---|---|---|
| ① | 돈은 수백만 원 나가는데 신환이 안 는다 | 노출·클릭 수치만 들이미는 대행사에 속 터짐 | 트래픽(노출)과 내원(전환)의 끊어진 연결고리 — "돈 버리는 마케팅 vs 환자가 찾아오는 마케팅" |
| ② | 의료법 위반 경고장이 무섭다 | 공격적 마케팅=영업정지 공포, 본인이 다 검수 | 의료광고 심의·의료법 완벽 이해 → "원장이 신경 안 써도 안전한 마케팅" |
| ③ | 우리 병원만의 차별성을 모르겠다 | 다들 친절·최신장비·경력만 말함 | 원장도 모르는 핵심 가치(USP) 발굴 기획력 |

## 3. 자가진단 4대 핵심 기능 (구현 설계)

### 기능 1. 플레이스 & 리뷰 최적화 진단 (지역 노출·환심도)
- 키워드 점유율: 핵심 키워드 노출 위치·경쟁 대비 점수 시각화 → **이미 구현**(플레이스+6영역 매트릭스)
- 리뷰 건강도: 기피 키워드(과잉진료·불친절·대기시간) 비율 경고
  → **구현 방식 주의**: 리뷰 본문 수집·감성분석은 법무 v2에서 보류(원본 미수집 원칙).
  대신 "리뷰 응대·수집 체크리스트 + 병원 직접 입력값" 방식으로 우회(합법).
- 전환 요소 체크: 사진(원장 이력·내부 전경)·찾아오는 길·네이버 예약 활성 → 체크리스트 점수화
- 인사이트 예: "상위 노출은 되는데 예약 동선·리뷰 관리 부실로 유입 단계 이탈"

### 기능 2. 의료법 위반 리스크 진단 (최강 리드 마그넷)
- 금지 표현 탐지: "최고·최초·완치·부작용 없음·100%·전후사진 무조건" 등 실시간 하이라이트
- 후기/사례 가이드라인 체크: 전후사진 조건(동일 조건·경과 기간 명시), 심의 대상 여부
- 리스크 등급: **[안전 / 주의 / 위험]** (숫자 확률 단정은 금지 — "위반 소지 높음"으로 표현)
- **구현 난이도 낮음·효과 최고**: 텍스트 붙여넣기 → 규칙 기반 필터. 서버 없이도 클라이언트 동작 가능
- 법무: 이 기능 자체가 "참고용·법적 자문 아님" 고지 필수. 최종 판단은 심의기구·법무.

### 기능 3. 마케팅 비용 효율(ROI)·누수 진단 계산기 (재무 관점)
- 입력: ① 월 마케팅비 ② 월 신환 수 ③ 월 방문자(조회수)
- 출력: **CPA**(신환 1인 획득비) = 마케팅비 / 신환. 업계 참고값과 비교.
  **CVR**(전환율) = 신환 / 방문자. 0.5% 미만 시 "노출은 되나 콘텐츠 설득력 부족" 경고.
- 인사이트: 화려한 노출 수에 속고 밑 빠진 독에 물 붓고 있었음을 재무적으로 자각
- **구현 난이도 낮음**: 순수 계산기(서버 불필요). 업계 참고값은 "참고·범위" 표기, 단정 금지.

### 기능 4. 환자 검색 여정(Search Journey) 그물망 진단
- 1단계 정보 탐색(증상 키워드, 예 "어깨가 안 올라가요") → 우리 콘텐츠 노출 Y/N
- 2단계 비교 분석(지역+치료법, 예 "OO동 오십견 잘하는 곳") → 노출 Y/N
- 3단계 최종 검증(병원명 브랜드) → 지식인·카페 악성 글 방치 여부 Y/N
- 인사이트: "2단계 지역 광고에만 돈 쓰고 1·3단계는 텅 비어 환자가 이탈"
- **구현**: 기존 auto_diagnose 키워드 자동 산출을 3단계로 재분류하면 대부분 재사용 가능

## 4. 채널별 원장의 속마음 vs 타격 전략

### [1단계 검색/노출]
| 채널 | 원장의 속마음 | 우리의 타격 전략 |
|---|---|---|
| 스마트플레이스 | "옆 병원은 1페이지, 우린 뒤. 대행사 순위 올림=불법 어뷰징? 걸리면 영구정지?" | 정공법 플레이스 SEO 점수화(체류·예약연동·톡톡응대·텍스트 가중치). "불법 트래픽은 삭제로 이어짐 → 합법적 Lock-in 전략" |
| 파워링크 | "클릭당 몇만 원, 부정클릭 당하는 듯, 돈만 쓰고 예약은 안 옴" | 대표 키워드 집착 탈피 → 세부 질환+치료법 키워드 재배치·시간대 입찰 최적화 + 부정클릭 IP 차단 |

### [2단계 신뢰/설득]
| 채널 | 원장의 속마음 | 우리의 타격 전략 |
|---|---|---|
| 홈페이지/브랜드 | "수백만 원 홈페이지, 보는 사람 있나? 뭐가 문제지?" | 7초 이탈 진단 — 첫 화면에 진료철학·USP 부재. 증상공감→치료프로세스→결과→네이버예약 단일 랜딩 |
| 블로그 | "2026년 심의 의무화 진짜? 대행사 글은 백과사전 같아 안 읽힘" | 심의 통과형 스토리텔링 대필 — 실제 케이스 각색, 심의필 기준 준수 샘플 동봉 |
| 유튜브 | "카메라 쑥스럽고 시간 없고 조회수 안 나옴" | 진료실 스마트폰 Q&A 쇼츠 위주. 기획·대본·편집 원스톱, 원장 시간 월 1시간만 |

### [3단계 바이럴/검증]
| 채널 | 원장의 속마음 | 우리의 타격 전략 |
|---|---|---|
| 카페/지식인 | "추천글 올리고 싶은데 바이럴 역풍·악플 무섭다" | 그레이 바이럴(가짜 아이디) 적발=처벌·정지 경고 → 실만족 고객 앰배서더·정보성 기부 콘텐츠 |
| 인스타/SNS | "정형/내과도 해야 하나? 이벤트 글 신고당할까 봐" | 과별 포지셔닝 — 비급여과: 비포애프터·이벤트 타깃광고 / 급여·질환과: 진료철학·건강꿀팁 카드뉴스 |
| 방문자 리뷰 | "별점 테러 무섭다. 가짜 영수증 리뷰 믿어도 되나?" | 가짜 리뷰=필터링 삭제·처벌 명시 → 자발 참여 동선 설계 + 악성 리뷰 대응 '명품 답글 템플릿' |

## 5. 진단 매트릭스 (진료과 선택 → 핵심 채널만 필터링)

원장이 진료과(예: 정형외과) 선택 시, 해당 과 핵심 채널 질문만 노출 → 전문성 극대화.

| 대분류 | 채널 | 자가진단 질문 | 진단 결과 오퍼 |
|---|---|---|---|
| 유입/노출 | 플레이스·파워링크 | 지역명+과목 검색 시 3위 안에 나오나? 광고비 대비 신환 경로를 추적하나? | 키워드 재배치 + 부정클릭 방어 |
| 설득/신뢰 | 블로그·홈페이지·유튜브 | 최근 콘텐츠가 2026 심의 기준 준수? 원장 진료철학 글이 있나? | 의료법 전수조사 + 스토리텔링 대필 샘플 |
| 평판/검증 | 카페·지식인·리뷰 | 악성 리뷰 대응 매뉴얼이 있나? 맘카페 평판 모니터링되나? | 리스크 CS 매뉴얼 + 커뮤니티 모니터링 |

## 6. DB 전환 시나리오 (진단 → 상담 신청)

1. **진입장벽 최소화**: 첫 3~4개 질문은 로그인·정보입력 없이 클릭(몰입 유도)
2. **결과 전 가치 제안(Lock-in)**: "종합 점수 58점(위험). 상세 리포트 + [맞춤 개선안 PDF] 받으시겠어요?"
3. **정보 입력**: 병원명·연락처(또는 카톡)·이메일
4. **백오피스 자동화**:
   - 원장 → 카톡 알림톡/이메일로 디자인된 리포트 즉시 발송
   - 베놈 영업팀 → "OO구 정형외과(58점·위험)" 실시간 인입 (관리자 신청관리 탭)
   - 후속: "의료법 위반 소지 높은 글 3개 발견, 무료 피드백" 미팅 제안

**법무 정합(v2 필수 준수)**:
- 두 단계 모두 무료. 광고성 수신 동의는 가입/열람 조건으로 강제 금지(선택 동의 유지).
- 콜드콜·콜드문자 금지. 신청·동의 채널 있는 원장에게만 연락. 동의 증적(일시·IP·문구버전) 저장.
- 마킹판/전체판 이중 산출은 **이미 구현**(auto_diagnose --html-masked). 회원 열람 인증은 접수 서버 결정 후.

## 7. 훅킹 카피 (홈페이지·리드마그넷 — 검토용, 심의 확인 후 사용)

- "매달 500만 원 쓰시는데 왜 신환은 지난달과 똑같을까요? 노출과 내원 사이의 끊어진 고리"
- "보건소 경고장 걱정 없는 안전한 병원 마케팅 — 원장이 검수하지 않아도 되는 정석"
- "경쟁 병원이 5개 생겨도 살아남는 우리 병원만의 차별화 전략"
- 진단형 콘텐츠 네이밍: "우리 병원 마케팅 밑 빠진 독 테스트", "의료법 위반 리스크 체크리스트"

## 8. 실행 로드맵 (v2 백로그 통합·재우선순위)

| 순위 | 항목 | 근거 | 상태/난이도 |
|---|---|---|---|
| **P0 즉시** | 의료법 리스크 진단(기능2) | 최강 리드마그넷·서버 불필요·규칙기반 | 신규·낮음 |
| **P0 즉시** | ROI 누수 계산기(기능3) | 재무 자각·순수 계산기 | 신규·낮음 |
| **P0 즉시** | 검색여정 3단계 뷰(기능4) | 기존 키워드 산출 재분류 | 개선·낮음 |
| P1 | 진단→DB 전환 랜딩 플로우(3~4클릭+Lock-in) | 전환 핵심 | 랜딩 개편·중 |
| P1 | 진료과 선택 → 채널 필터 매트릭스 | 전문성 극대화 | 중 |
| P2(서버 후) | 셀프 진단 웹 실행 + 알림톡 자동발송 | 접수 서버 필요 | 중 |
| P2 | 월간 변화 리포트 | 재방문 동기 | 중 |
| 상시 | 리뷰 응대·의료법 체크리스트 콘텐츠 | 리드마그넷 보강 | 낮음 |

## 9. 리스크 관리 (이 도구가 스스로 지켜야 할 것)

- 의료법 진단 결과에 "법적 자문 아님·참고용" 고지 (오판 시 책임 한계)
- CPA·CVR 업계 참고값은 "범위·추정" 표기, 단정 금지
- "보건소 신고 확률 80%" 같은 수치 단정 → "위반 소지 높음(위험 등급)"으로 순화
- 리뷰 감성분석은 본문 미수집 원칙(v2) 준수 — 체크리스트·직접입력 우회
- 공포 마케팅 톤 억제 — 사실 고지로 위기감 전달, 과장 배제
