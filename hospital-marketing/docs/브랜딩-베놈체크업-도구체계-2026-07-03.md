# 브랜딩 기획 — 베놈 체크업 (VENOM Checkup) · 도구 체계 정리

목적: 베놈이 자체 개발한 무료 점검 도구들을 하나의 **부르기 쉬운 브랜드**로 묶고,
개별 도구 이름을 통일한다. 계기: 도구가 늘어나며 "이게 무슨 화면인지" 혼란(근거자료함 피드백)이
반복 → 이름·구조·비유를 한 번에 정리.

## 1. 상위 브랜드: 베놈 체크업 (VENOM Checkup)

**슬로건: "우리 병원 마케팅, 건강검진 받아보세요."**

선정 이유:
- 핵심 고객 = 병원 원장. 그들의 언어(진단·검진·소견·처방)와 100% 일치.
- 무료 도구 + 이미지 제고 목적(정책 v2)에 "무료 건강검진" 프레임이 정확히 맞음.
- 자연스러운 퍼널: **무료 검진 → 소견서 → (원하면) 처방 = 베놈 대행**. 영업이 아니라 진료.
- 소상공인·마케터에게도 "검진"은 위화감 없이 통함(대안 후보 "레이더/닥터베놈"보다 범용).

> 이름은 코드/URL에 `checkup`으로 반영하되, 화면 표기명은 상수 1곳에서 관리해
> 추후 브랜드명 변경 시 전 화면 일괄 교체 가능하도록 한다.

## 2. 도구 명칭 체계 (검진 항목 비유)

| 구분 | 브랜드 도구명 | 실제 기능 | 파일 | 상태 |
|---|---|---|---|---|
| 이용자 | **원클릭 검진** | 업체명 하나로 자동 진단(마킹판/전체판) | pipeline/auto_diagnose | ✅ |
| 이용자 | **종합검진** | 노출·경쟁·키워드 종합 대시보드 | index.html | ✅ |
| 이용자 | **리스크 진단** | 의료법 위반 위험 자가점검 | self-check(리스크) | 🆕 이번 |
| 이용자 | **비용 진단** | 광고비 효율(CPA/CVR) 계산 | self-check(비용) | 🆕 이번 |
| 이용자 | **동선 진단** | 환자 검색여정 3단계 점검 | self-check(동선) | 🆕 이번 |
| 이용자 | **입지 진단** | 개원 입지 비교 분석 | location/ | ✅ |
| 이용자 | **심의 챗봇** | 의료광고심의 Q&A·문구 자가진단 (RAG AI) | 베놈 사이트 `chatbot/demo.html` (외부 링크) | ✅ 2026-07-06 허브 등재 |
| 접수 | **검진 신청** | 무료 진단 신청 랜딩 | landing/ | ✅ |
| 내부 | **차트룸** | 관리자단(신청·상세·어뷰징·근거자료함) | admin/ | ✅ |

명명 규칙: 이용자 도구는 **"OO 진단/검진"** 2음절+진단 형태로 통일 → 부르기 쉽고 검진 비유 유지.

## 3. 검진 리포트 등급 언어 (통일)

모든 도구가 같은 등급 언어를 쓴다(혼란 방지, 법무 정합):
- **양호 / 주의 / 위험** 3단계 (숫자 확률 단정 금지 — "위반 소지 높음(위험)" 식)
- 색: 양호=녹색, 주의=주황, 위험=적색 (검증 팔레트 status 색 재사용)
- 종합 점수는 "검진 점수(당사 산식·0~100)"로 표기, "순위/보장" 표현 금지

## 4. 퍼널 내 위치

```
[홈페이지/블로그 유입]
      │
      ▼
  베놈 체크업 (무료 검진 5종)
   ├ 원클릭 검진 ─── 업체명만 → 즉시 마킹판 소견
   ├ 종합검진 ────── 노출·경쟁 종합
   ├ 리스크 진단 ─── 의료법 위험 (최강 훅)
   ├ 비용 진단 ───── 광고비 낭비 자각
   └ 동선 진단 ───── 유입 경로 구멍
      │  결과 → "정밀 소견서 + 개선안 PDF 받기"
      ▼
  검진 신청(랜딩) ── 필수 동의만으로 접수
      │  차트룸에 실시간 인입(점수·등급)
      ▼
  베놈 상담(=처방) → 광고대행(=치료)
```

## 5. 로고·비주얼 톤 (가이드)

- 심볼: 기존 각진 "V" 마크 유지 + 체크업은 "V + 체크(✓)/맥박선" 결합 여지
- 색: 검증 팔레트(주색 #2a78d6) 유지 — 신뢰의 파랑
- 톤: 병원 UI답게 **차분·정확·근거 중심**. 공포 마케팅·과장 금지(정책 v2·v3).
- 아이콘: 검진 항목별 픽토그램(종합=청진기, 리스크=경고, 비용=지폐, 동선=경로, 입지=핀)

## 6. 적용 순서

1. (이번) P0 도구 3종을 브랜드명으로 신규 구현 → self-check 페이지
2. 기존 화면 헤더/타이틀에 "베놈 체크업" 상위 브랜드 병기(상수 1곳 관리)
3. 도구 간 이동 허브(체크업 홈) 신설 검토 — 5종 도구 카드로 진입
4. 검진 신청·차트룸 명칭을 화면에 반영(관리자단 탭·랜딩 제목)
