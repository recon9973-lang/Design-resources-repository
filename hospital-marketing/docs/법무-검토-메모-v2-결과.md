# (주)베놈 병원 검색정보 진단 프로그램 v2 법무 검토 메모

검토일: 2026-07-02  
대상 파일: `TalkFile_법무검토설명자료v2베놈.pdf.pdf`  
검토 대상 모델: 병원 전문 광고대행사인 (주)베놈이 병원에 무료 검색정보 진단 리포트를 제공하고, 이후 선택적으로 광고대행 계약을 체결하는 구조

> 본 문서는 사업 판단을 위한 법무 검토 초안입니다. 실제 외부 배포·영업 개시 전에는 변호사의 최종 검토와 네이버 API 약관 확인이 필요합니다.

## 1. 핵심 결론

v2는 v1보다 데이터 리스크는 낮아졌지만, 영업·광고·개인정보 리스크는 더 선명해졌습니다. 유료 SaaS 판매를 포기하고 무료 진단자료로 전환한 점은 결제·구독 약관 리스크를 줄입니다. 그러나 무료 리포트가 광고대행 계약을 유도하는 영업 도구라면, 법적으로는 여전히 "영리 목적" 활동입니다.

결론은 다음과 같습니다.

| 쟁점 | 판단 | 핵심 조건 |
|---|---|---|
| 무료 진단 리포트의 의료광고 해당성 | 조건부 낮음 | 환자 비노출, 병원 내부 참고용, 광고물 전재 금지, 특정 병원 홍보문구 미포함 |
| 표시광고법상 광고 해당성 | 조건부 중간 | 베놈 서비스 홍보·리드젠 자료로는 광고성 있음. 무료·성과·진단 정확성 고지 필요 |
| 리포트와 광고물 책임 분리 | 조건부 허용 | 단계·파일·계약·승인 절차를 분리하고 광고물에는 리포트 수치 전재 금지 |
| 병원의 무단 전재 시 베놈 책임 | 관리 여부에 따라 중간 | 제공조건 동의, 워터마크, 로그, 발견 시 중단·삭제요청 프로세스 필요 |
| 신청폼 기반 리드젠 | 조건부 허용 | 개인정보 수집·이용 동의와 마케팅/광고성 정보 수신 동의를 분리 |
| 콜드 문자·메일 | 원칙 금지에 가까움 | 명시적 사전 동의 없으면 위험. 기존 거래처 6개월 예외 등만 제한적 가능 |
| 공개 대표번호 전화 | 제한적 가능 | 문자·메일보다 여지는 있으나 수집출처 고지, 거부등록, 녹취/로그 필요 |
| 네이버 API 데이터 활용 | v1과 동일하게 조건부 허용 | 공식 API 범위, 출처/기준일 표시, "순위" 표현 금지 |
| 검색화면 수작업 스냅샷 | 보류 유지 | 내부 법무검증 외 운영 금지 권고 |
| 병원 직접 입력 리뷰/사진 수 | 조건부 허용 | 병원이 자기 관리화면에서 직접 확인·입력. 베놈 자동/대량 수집 금지 |

## 2. v1 대비 법적 의미가 달라진 부분

v1은 "병원에 판매하는 유료 SaaS"였습니다. v2는 "무료 배포 + 광고대행사 영업/상담 도구"입니다. 이 변경으로 줄어든 리스크와 새로 생긴 리스크는 분명합니다.

줄어든 리스크:

- 결제정보, PG 위탁, 구독 해지, 환불, 전자상거래성 약관 리스크가 사라졌습니다.
- 고객에게 월 구독 상품으로 파는 데이터 서비스라는 인상은 약해졌습니다.
- 실제 검색화면 스냅샷과 플레이스 자동 수집을 보류한 것은 매우 좋은 조치입니다.

새로 커진 리스크:

- 무료 리포트가 광고대행 계약 유인을 위한 영업자료가 되므로 정보통신망법상 광고성 정보, 표시광고법상 무료·성과 표시, 개인정보 마케팅 동의가 핵심 리스크가 됩니다.
- 베놈은 단순 소프트웨어 제공자가 아니라 의료광고 제작·집행 대행사입니다. 리포트 내용이 후속 광고물에 흘러들어가면 의료광고 공동 관여 책임이 문제될 수 있습니다.
- 콜드 아웃리치가 포함되어 문자·메일·전화 채널별 규제가 직접 적용됩니다.

## 3. 법조문 기준

### 3.1 의료법

의료법 제56조는 의료기관 개설자, 의료기관의 장 또는 의료인이 아닌 자의 의료광고를 금지하고, 의료인등에게도 치료경험담, 거짓·과장, 다른 의료인등의 기능 또는 진료방법과 비교하는 광고, 비방 광고, 미심의 광고 등을 금지합니다.

의료법 제57조는 일정 매체의 의료광고에 대해 사전 심의를 요구합니다. 의료법 시행령 제24조는 심의 대상 인터넷 매체를 구체화하며, 전년도 말 기준 직전 3개월간 일일 평균 이용자 수 10만명 이상인 정보통신서비스 제공자가 운영하는 인터넷 매체와 일정 SNS 광고매체를 포함합니다.

v2 적용:

- 리포트가 병원 내부에서만 열람되고 환자에게 노출되지 않는다면 의료법상 의료광고로 볼 가능성은 낮습니다.
- 베놈이 리포트 내용을 광고문안, 랜딩페이지, SNS, 블로그, 플레이스 소개글, 병원 홈페이지, 전단, 배너 등에 전재하거나 가공해 환자에게 보이면 그 순간 의료광고 규제가 적용됩니다.
- 광고대행사인 베놈은 의료인등이 아니므로, 베놈 명의로 특정 병원의 의료행위·의료기관·의료인을 소비자에게 알리는 행위는 특히 위험합니다. 광고물은 병원 명의, 병원 승인, 필요 시 심의 절차를 거쳐야 합니다.

### 3.2 표시광고법

표시·광고의 공정화에 관한 법률 제3조는 소비자를 속이거나 잘못 알게 할 우려가 있고 공정거래질서를 해칠 우려가 있는 거짓·과장, 기만, 부당 비교, 비방 표시·광고를 금지합니다.

v2 적용:

- 무료 진단 리포트는 병원 소비자에게 제공되는 영업 유인입니다. 병원은 베놈 광고대행 서비스의 잠재 고객이므로, 베놈의 무료 진단 안내 페이지·문자·메일·상담자료는 표시광고법상 광고로 평가될 수 있습니다.
- "무료"는 실제로 무료여야 합니다. 진단을 받으려면 상담 필수, 계약 유도 필수, 광고대행 계약 조건부 무료라면 명확히 표시해야 합니다.
- "상위노출 가능성 진단", "매출 개선", "환자 유입 증가"와 같은 문구는 객관적 근거가 없으면 위험합니다.

### 3.3 개인정보보호법

개인정보보호법 제15조는 개인정보 수집·이용 근거와 동의 시 고지사항을 정합니다. 제22조는 동의 사항을 구분해 명확히 알리고, 특히 재화나 서비스 홍보·판매 권유를 위한 개인정보 처리 동의는 구분해서 받아야 한다고 정합니다. 제30조는 개인정보 처리방침 수립·공개 의무를 둡니다. 제26조는 개인정보 처리 위탁 시 문서화, 수탁자 공개, 감독 의무를 둡니다.

v2 적용:

- 원장/담당자 성명, 휴대전화, 이메일은 개인정보입니다.
- "무료 리포트 제공" 목적과 "컨설팅·마케팅 정보 안내" 목적은 분리해야 합니다.
- 마케팅 동의가 없어도 무료 리포트 자체는 제공해야 합니다. 개인정보보호법 제22조 제5항 취지상 선택 동의를 거부했다는 이유로 기본 서비스를 거부하면 위험합니다.
- 병원 대표번호·대표 이메일이라도 특정 원장 또는 담당자 개인에게 연결되면 개인정보로 취급하는 것이 안전합니다.

### 3.4 정보통신망법상 광고성 정보

정보통신망 이용촉진 및 정보보호 등에 관한 법률 제50조는 전자적 전송매체를 이용한 영리목적 광고성 정보 전송에 명시적 사전 동의를 요구합니다. 수신거부 또는 철회가 있으면 전송할 수 없고, 오후 9시부터 다음 날 오전 8시까지 전송하려면 별도 동의가 필요합니다. 전송자 명칭·연락처와 수신거부·철회 방법도 표시해야 합니다.

시행령 제61조는 기존 거래관계 예외 기간을 거래 종료 후 6개월로 정합니다. 시행령 제62조는 수신거부 또는 철회용 무료전화서비스 등을 제공하도록 하고, 제62조의3은 2년마다 수신동의 여부를 확인하도록 합니다.

v2 적용:

- 무료 진단 리포트 안내, 후속 상담 안내, 광고대행 제안은 베놈의 영업으로 연결되므로 영리목적 광고성 정보입니다.
- 신청폼에서 명시적으로 광고성 정보 수신 동의를 받은 병원에만 문자·메일을 보내는 구조가 안전합니다.
- 신규 병원 콜드 문자·메일은 원칙적으로 금지 의견입니다.
- 기존 거래처는 같은 종류의 서비스에 관해 거래 종료 후 6개월 이내라는 예외가 있을 수 있으나, 수신거부자는 제외해야 합니다.

### 3.5 데이터·크롤링 관련 법

저작권법 제93조는 데이터베이스제작자의 권리를 보호하고, 개별 소재라도 반복적·체계적으로 이용해 데이터베이스의 일반적 이용과 충돌하거나 제작자 이익을 부당하게 해치면 상당 부분 이용으로 볼 수 있습니다.

부정경쟁방지법 제2조 제1호 카목은 접근권한 없는 데이터 취득, 접근권한을 넘은 부정 사용, 기술적 보호조치 무력화 등을 데이터 부정사용으로 정합니다.

형법 제314조는 정보처리장치에 허위 정보 또는 부정한 명령을 입력하거나 기타 방법으로 정보처리에 장애를 발생시켜 업무를 방해하는 행위를 처벌합니다.

v2 적용:

- 네이버 실제 검색화면 스냅샷과 플레이스 자동 수집 보류는 타당합니다.
- 병원이 자기 플레이스 관리화면에서 직접 확인해 입력하는 방식은 자동 수집보다 훨씬 안전합니다.
- 베놈 직원이 대신 수기로 확인해 입력하는 것은 자동 스크래핑보다는 낮지만, 반복·체계화되면 제3자 플랫폼 데이터 수집으로 평가될 여지가 있습니다.

### 3.6 공공데이터법

공공데이터의 제공 및 이용 활성화에 관한 법률 제3조는 공공데이터의 영리적 이용을 원칙적으로 금지·제한해서는 안 된다는 방향을 둡니다. 다만 이용자는 법령과 이용조건을 준수하고 타인의 권리를 침해하지 않아야 합니다.

v2 적용:

- HIRA 병원정보서비스, 통계청 SGIS, 공공데이터 CSV 활용 자체는 조건부 허용 가능성이 높습니다.
- 출처, 기준일, 갱신주기, 오류 가능성, 당사 산식이라는 점을 표시해야 합니다.

## 4. 질의별 검토 의견

### A-1. 무료 진단 리포트의 법적 성격

의견: 조건부 허용.

환자에게 노출되지 않고 병원 원장·마케팅 담당자에게 내부 참고용으로 제공되는 리포트라면 의료법상 의료광고 가능성은 낮습니다. 무료라는 점 자체가 의료광고성을 낮추는 핵심은 아닙니다. 핵심은 "수신자"와 "내용"입니다. 환자 소비자에게 의료기관·의료행위·의료인을 알리는 자료인지, 또는 병원 내부 운영자료인지가 중요합니다.

다만 표시광고법 관점에서는 베놈의 광고대행 서비스를 홍보하는 리드젠 자료입니다. 따라서 무료 진단 안내 페이지와 신청폼에는 다음 고지가 필요합니다.

- 무료 진단은 병원 내부 운영 참고자료입니다.
- 특정 검색 순위, 환자 유입, 매출 증가를 보장하지 않습니다.
- 진단 결과는 당사 산식에 따른 참고 지표입니다.
- 후속 광고대행 계약은 선택 사항이며, 진단 신청과 별개입니다.

### A-2. 컨설팅·광고 제작의 책임 경계

의견: 조건부 허용. 현재 내규는 방향이 맞지만 보완 필요.

리포트 단계와 광고물 단계의 책임 분리는 문서상으로만 두면 부족합니다. 파일, 프로세스, 권한, 승인 기록까지 분리해야 합니다.

필수 보완:

- 리포트 파일명/표지/푸터에 "광고물 아님" 표시.
- 리포트 내 경쟁 병원명·점수·API 등급은 광고물 자동전재 금지 필드로 지정.
- 광고물 제작 브리프에는 리포트 수치가 자동 복사되지 않도록 시스템 차단.
- 광고물 제작 시 별도 체크리스트: 의료법 제56조 금지표현, 심의 대상 매체, 병원 승인자, 심의번호/승인일.
- 광고물은 병원 명의로 심의·게시. 베놈 단독 명의 게시 금지.

### A-3. 병원의 무단 전재에 대한 베놈 책임

의견: 조건부 관리 필요.

병원이 고지를 무시하고 리포트를 환자 대상 광고에 전재한 경우, 베놈이 이를 알지 못했고 전재를 유도하지 않았으며 합리적 방지조치를 했다면 책임 가능성은 낮아집니다. 반대로 베놈이 광고에 쓰기 쉬운 문구·이미지·점수 배지를 제공하거나, 병원의 전재 사실을 알고도 방치하면 방조 또는 공동 관여 리스크가 커집니다.

필수 조치:

- 제공 전 클릭 동의: "환자 대상 광고·홍보물 전재 금지".
- PDF 워터마크: "병원 내부 참고용 / 광고 전재 금지".
- 경쟁 병원명과 점수는 복사 방지 또는 별도 페이지 분리.
- 고객 약정에 위반 시 이용중지, 자료 회수, 정정·삭제 요청 권한.
- 베놈이 무단 전재를 발견하면 즉시 삭제 요청, 재발 방지 안내, 기록 보관.

### A-4. 리드젠 개인정보·광고성 정보

의견: 신청폼 기반은 조건부 허용, 콜드 문자·메일은 원칙 금지.

신청폼 동의만으로 충분하려면 동의를 쪼개야 합니다.

필수 체크박스:

- 필수: 개인정보 수집·이용 동의 - 무료 진단 리포트 제공 목적.
- 선택: 개인정보 수집·이용 동의 - 컨설팅 및 광고대행 상담 안내 목적.
- 선택: 광고성 정보 수신 동의 - 문자.
- 선택: 광고성 정보 수신 동의 - 이메일.
- 선택: 광고성 정보 수신 동의 - 전화 상담.

기록 보관:

- 동의 일시, IP, User-Agent, 동의 문구 버전, 수신 채널, 철회 일시를 보관합니다.
- 광고성 정보 수신 동의는 2년마다 재확인합니다.
- 수신거부자는 별도 suppression list로 관리하고 어떤 캠페인에서도 제외합니다.

콜드 아웃리치:

- 공개 이메일·대표번호라도 문자·이메일로 무료 진단/상담을 제안하면 영리목적 광고성 정보로 볼 가능성이 높습니다.
- 명시적 사전 동의 없는 콜드 문자·메일은 금지 의견입니다.
- 대표번호로의 1차 전화는 문자·메일보다 여지는 있으나, 수집출처 고지, 통화 목적 고지, 거부 의사 즉시 반영, 재전화 금지, 녹취/로그가 필요합니다.

### A-5. 무료 배포 시 필요한 최소 문서 세트

의견: 유료 약관 대신 4종 문서 필요.

1. 무료 진단자료 제공조건
2. 개인정보 수집·이용 동의서
3. 광고성 정보 수신 동의서
4. 개인정보처리방침

제공조건 필수 문구:

- 본 리포트는 병원 내부 운영 참고자료이며 의료광고물이 아닙니다.
- 환자 대상 광고·홍보물로 전재, 캡처, 인용, 배포할 수 없습니다.
- 특정 검색 순위, 환자 유입, 매출 증가를 보장하지 않습니다.
- 진단 점수와 등급은 당사 산식에 따른 참고 지표입니다.
- 공공데이터 및 외부 API의 오류·변경·중단 가능성이 있습니다.
- 후속 광고대행 계약은 선택 사항입니다.

### B-6. 네이버 지역 검색 API 응답 저장·가공·표시

의견: v1과 동일하게 조건부 허용.

네이버 지역 검색 API 공식 문서는 지역 검색 결과를 XML/JSON으로 제공하는 오픈 API입니다. 문서상 `display` 최댓값은 5, `start` 최댓값은 1로 안내됩니다. 따라서 제품 표시는 "네이버 지역 검색 API 기준 상위 5건 내 포함 여부"로 한정해야 합니다.

금지 표현:

- 네이버 순위
- 플레이스 순위
- 실시간 순위
- 상위노출 보장
- 매출 상승 보장

권장 표현:

- 네이버 지역 검색 API 기준 상위 5건 내 노출 여부
- 측정일 기준 API 응답 내 포함 여부
- 당사 산식에 따른 검색정보 운영 진단 등급

### B-7. 월 1회 직원 수작업 검색화면 확인

의견: 기능 보류 유지. 내부 법무검증용 예외만 제한적으로 검토.

직원 수작업 확인은 자동 크롤링보다 위험이 낮지만, 베놈이 반복적으로 검색화면을 확인·저장하고 이를 유료 광고대행 영업 근거로 체계화하면 플랫폼 데이터 활용 리스크가 남습니다.

조건을 둔다면:

- 고객 제공 리포트에는 반영하지 않습니다.
- HTML 저장 금지, 원본 화면 이미지 장기 보관 금지.
- 내부 법무검증 목적, 월 1회 이하, 지정 직원, 비로그인, 개인화 결과 미사용.
- 차단·캡차·경고가 있으면 즉시 중단.

### B-8. 플레이스 리뷰 수·사진 수 입력

의견: 병원 직접 입력은 조건부 허용, 베놈 직원 수기 입력은 보류 또는 고위험 관리.

병원이 자기 플레이스 관리화면 또는 공개 화면에서 수치를 확인해 직접 입력하는 방식은 가장 안전합니다. 이 경우 베놈은 제3자 플랫폼에서 데이터를 수집하는 자가 아니라 병원 입력값을 처리하는 자에 가깝습니다.

베놈 직원이 대신 입력하면:

- 수작업이라도 반복·체계적이면 플랫폼 데이터 수집으로 평가될 수 있습니다.
- 리뷰 본문, 작성자, 환자 치료경험담은 절대 수집하면 안 됩니다.
- 꼭 필요하다면 병원 위임 확인서, 수기 확인 로그, 월 1회 이하, 숫자만 입력, 원본 저장 금지 조건이 필요합니다.

### B-9. 1인 의원 상호에 원장 성명 포함

의견: 개인정보 가능성을 전제로 처리.

병원명·주소는 사업장 정보이지만, 1인 의원의 상호에 원장 성명이 포함되면 특정 개인을 식별할 수 있습니다. 공개 공공데이터라고 해도 개인정보 가능성을 전제로 최소 처리해야 합니다.

조치:

- 원장 성명 자체를 별도 필드로 추출하지 않습니다.
- 공공데이터 원문 표시 범위로 제한합니다.
- 정정·삭제·비노출 요청 창구를 둡니다.
- 검색엔진 색인되는 공개 페이지에는 1인 의원명과 진단 점수를 함께 노출하지 않습니다.

### B-10. 공공데이터+API 결합 진단 점수

의견: 조건부 허용.

무료 자료에서도 표시광고 쟁점은 생길 수 있습니다. 병원이 베놈 광고대행 서비스를 선택하게 만드는 자료이기 때문입니다.

필수 고지:

- 진단 점수는 당사 산식에 따른 참고 지표입니다.
- 점수는 의료서비스의 질, 치료 효과, 환자 만족도, 매출 가능성을 의미하지 않습니다.
- 외부 API와 공공데이터의 기준일·측정 조건에 따라 변동될 수 있습니다.
- 경쟁 병원에 대한 우열 판단이나 비방 목적으로 사용할 수 없습니다.

## 5. 콜드 아웃리치 운영 기준

### 문자·이메일

원칙: 사전 동의 없으면 보내지 않습니다.

허용 가능한 경우:

- 신청폼에서 해당 채널 광고성 정보 수신에 명시적으로 동의한 경우.
- 기존 거래관계가 있고, 거래 종료 후 6개월 이내이며, 같은 종류의 재화·서비스 안내이고, 수신거부 이력이 없는 경우.

전송 시 필수:

- 제목 또는 본문에 광고성 정보임을 명확히 표시합니다.
- 전송자 명칭과 연락처를 표시합니다.
- 무료 수신거부 방법을 제공합니다.
- 야간 전송 금지. 이메일은 시행령상 별도 야간동의 예외가 있으나 실무상 야간 발송은 피하는 것이 안전합니다.
- 수신거부 처리 결과를 통지합니다.

### 전화

원칙: 대표번호 대상 1차 안내는 문자·이메일보다 여지가 있으나, 공격적 영업은 금지.

체크리스트:

- 전화 시작 시 회사명, 담당자명, 연락처 수집출처, 목적을 고지합니다.
- "무료 진단자료 안내 및 광고대행 상담 가능 여부 확인"이라고 말합니다.
- 거부 의사를 표시하면 즉시 통화 종료하고 재연락 금지 목록에 등록합니다.
- 개인 휴대전화로 콜드콜은 하지 않습니다.
- 통화기록, 거부기록, 담당자, 일시를 남깁니다.

## 7. 판례·사례 검토

### 7.1 직접 적용 가능한 판례

이번 검토 범위에서 "광고대행사가 병원에 무료 내부 진단 리포트를 제공하고 후속 광고대행 계약을 체결하는 구조"와 동일한 사실관계의 공개 판례는 직접 확인하지 못했습니다. 따라서 본 검토는 조문, 제도 취지, 유사 리스크 구조를 기준으로 판단합니다.

### 7.2 의료광고 사전심의 관련 헌법재판소 결정

의료법 제56조와 제57조의 현행 조문 자체가 과거 의료광고 사전심의 제도에 대한 헌법재판소 위헌 결정 이후 2018년 개정된 흐름을 반영합니다. 현재는 자율심의기구 중심의 사전심의 구조이며, 심의 대상 매체에 광고물을 게시할 때는 의료법 제57조와 시행령 제24조를 기준으로 별도 검토해야 합니다.

v2에의 의미:

- 리포트 자체가 내부자료로 머물면 심의 대상 의료광고가 아닐 가능성이 높습니다.
- 리포트 내용이 병원 홈페이지, 블로그, SNS, 포털 광고, 랜딩페이지 등 환자 대상 매체에 들어가면 심의 여부와 금지광고 여부를 별도 판단해야 합니다.

### 7.3 크롤링·데이터 무단 이용 사례의 일반 법리

국내외 크롤링 분쟁에서 반복적으로 문제 되는 요소는 기술적 보호조치 회피, 대량·반복 접근, 약관 위반, 원 데이터베이스 대체성, 서버 부하, 경고 후 계속 접근입니다. v2가 실제 검색화면 자동 캡처와 플레이스 자동 수집을 포기한 것은 이 리스크를 실질적으로 낮추는 조치입니다.

실무상 적용:

- 공식 API와 공공데이터만 자동 수집합니다.
- 화면 HTML 파싱, 자동 캡처, 로그인 세션 활용, 캡차 우회는 금지합니다.
- 병원 직접 입력 방식은 유지하되, 베놈 대리 입력은 예외 승인제로 둡니다.

## 8. 출시 전 필수 조치

2. 개인정보 수집·이용 동의와 마케팅 동의를 분리.
3. 광고성 정보 수신 동의를 문자, 이메일, 전화 채널별로 분리.
4. 수신거부 DB와 재연락 금지 목록 구축.
5. 리포트와 광고물 제작 프로세스를 시스템상 분리.
6. 광고물 제작 체크리스트에 의료법 제56조 금지사항과 제57조 심의 여부 추가.
7. 리포트 내 경쟁 병원명·점수·등급의 광고물 전재 금지 기술통제.
8. 네이버 실제 검색화면 스냅샷과 플레이스 자동 수집은 계속 보류.
9. 공공데이터/네이버 API 출처, 기준일, 측정조건 표시.
10. 1인 의원 상호 정정·삭제 요청 창구 마련.

## 9. 최종 판단

v2는 출시 방향이 더 현실적입니다. 무료 진단 리포트를 "병원 내부 운영 참고자료"로 제한하고, 데이터 수집을 공식 API와 공공데이터로 묶고, 검색화면 스냅샷과 플레이스 자동 수집을 포기한 점은 법무적으로 좋은 수정입니다.

그러나 베놈이 광고대행사라는 점 때문에 책임 경계는 더 엄격하게 잡아야 합니다. 무료 리포트가 광고대행 계약의 영업 도구인 이상, "무료니까 규제가 약하다"가 아니라 "무료지만 영리 목적 광고·개인정보·의료광고 전단계 규제가 붙는다"가 정확한 판단입니다.

운영 결론:

- 신청 기반 무료 진단: 조건부 허용.
- 기존 거래처 대상 안내: 조건부 허용.
- 신규 병원 콜드 문자·메일: 금지 권고.
- 대표번호 대상 1차 전화: 제한적 허용, 거부관리 필수.
- 리포트 내부 제공: 조건부 허용.
- 리포트 내용 광고물 전재: 금지.
- 후속 광고대행: 별도 계약·병원 승인·심의 검토를 거치면 가능.
