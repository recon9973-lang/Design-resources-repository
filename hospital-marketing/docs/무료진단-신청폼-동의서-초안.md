## 1. 무료 진단 신청 페이지 본문

### 페이지 제목

### 안내 문구

(주)베놈은 병원의 공개정보와 검색정보 운영 상태를 점검하기 위해 무료 진단 리포트를 제공합니다.

본 리포트는 병원 내부 운영 참고자료이며 의료광고물이 아닙니다. 특정 검색 순위 달성, 환자 유입, 예약 증가 또는 매출 증가를 보장하지 않습니다.

진단 결과는 공공데이터, 네이버 지역 검색 API, 통계청 공개 지표 및 당사 산식에 따른 참고 자료입니다. 후속 광고대행 계약은 선택 사항이며, 무료 진단 신청과 별개입니다.

6. 후속 광고대행 계약은 선택 사항이며, 무료 진단 신청만으로 광고대행 계약이 체결되지 않습니다.
