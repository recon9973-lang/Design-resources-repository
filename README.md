
별도의 빌드 과정 없이 `index.html` 하나로 동작합니다.

## 🚀 GitHub Pages 켜는 법

1. GitHub에서 이 저장소로 이동합니다.
2. 상단의 **Settings** 탭을 클릭합니다.
3. 왼쪽 사이드바에서 **Pages** 메뉴를 선택합니다.
4. **Build and deployment → Source** 를 **Deploy from a branch** 로 설정합니다.
5. **Branch** 에서 배포할 브랜치(예: `main`)와 폴더 `/ (root)` 를 선택한 뒤 **Save** 를 누릅니다.
6. 잠시(1~2분) 기다리면 페이지 상단에 공개 주소가 표시됩니다.

## 🛠 로컬에서 미리보기

```bash
# 저장소를 클론한 뒤
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```
