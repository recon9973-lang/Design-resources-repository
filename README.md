## 🛠 사용 방법 (Usage)

- `index.html` 단일 파일로 구성된 정적 페이지입니다. 별도의 빌드 과정이 필요 없습니다.
- 로컬에서 보려면 `index.html`을 브라우저로 열거나, 간단한 정적 서버를 실행하세요:

```bash
python3 -m http.server 8000
# http://localhost:8000 접속
```
